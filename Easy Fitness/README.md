# Анализатор Калорий в Блюде

Это простое веб-приложение для подсчета калорий, белков и жиров в блюде с помощью фотографии или камеры.

## Функции
- 📷 Доступ к камере устройства
- 📁 Загрузка фотографии
- 🍎 Анализ питательных веществ (симулированный)
- 📊 Шкала прогресса по дневной цели калорий

## Как использовать
1. Откройте `index.html` в браузере.
2. Выберите цель: похудение или набор массы.
3. Введите дневную цель калорий.
4. Используйте камеру или загрузите фото блюда.
5. Посмотрите результаты и прогресс.

## Технические детали
- HTML, CSS, JavaScript
- Доступ к камере через `getUserMedia`
- Симуляция анализа (для реального использования нужен API, например, Nutritionix)

## Запуск
Просто откройте `index.html` в современном браузере с поддержкой камеры.
